# machine-learning-101

Collection of introductory machine learning projects in Python. Currently includes a Titanic survival prediction exploration (Kaggle starter dataset).

## Stack

Python 3, scikit-learn, Pandas, NumPy, Seaborn, Matplotlib

## Projects

- **Titanic** — Survival prediction using the classic Kaggle dataset (`train.csv`, `test.csv`). Includes data exploration, feature engineering, and train/test split.

## Usage

```bash
pip install -r requirements.txt
jupyter notebook
```
